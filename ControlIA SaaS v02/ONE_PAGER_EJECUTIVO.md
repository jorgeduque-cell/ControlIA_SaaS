## 🎯 PRIMEROS PASOS (HOY)

```bash
# 1. Clonar (2 min)
git clone https://github.com/VoltAgent/voltagent.git openclaw-solo

# 2. Setup (10 min)
cd openclaw-solo && npm install

# 3. Variables (5 min)
cp .env.example .env.local
# Agregar: OPENAI_API_KEY, SUPABASE_URL, WHATSAPP_TOKEN

# 4. Deploy (3 min)
vercel --prod
```

**Total: 20 minutos para tener base en producción**

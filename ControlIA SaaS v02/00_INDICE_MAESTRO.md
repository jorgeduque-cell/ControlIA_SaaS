## 🎯 STACK TECNOLÓGICO CONSOLIDADO

### Frontend
- React 18 + TypeScript 5.3 + Tailwind CSS + Vite

### Backend
- Node.js 20 + Fastify + GraphQL + tRPC
- Python (para ML/AI workloads)

### Infraestructura
- AWS São Paulo (sa-east-1) + N. Virginia (DR)
- Kubernetes (EKS) + Istio Service Mesh
- Terraform + ArgoCD (GitOps)

### Bases de Datos
- PostgreSQL 16 (Aurora) + Redis 7 + MongoDB Atlas
- Pinecone/pgvector (vector DB)
- Kafka (event streaming)

### IA/ML
- OpenAI GPT-4 + Claude 3.5 + Llama 3
- LiteLLM (router inteligente)
- Langfuse (observabilidad)

### DevOps
- GitHub Actions (CI/CD)
- Prometheus + Grafana + Loki + Tempo
- LaunchDarkly (feature flags)
